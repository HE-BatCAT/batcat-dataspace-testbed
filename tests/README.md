# Tests for token exchange and transfer of data

## Requirements

* Running testbed [see ../README.md](../README.md)
* Python 3
* POSIX shell

## Installation

* `python -m venv .venv`
* `source .venv/bin/activate`
* `pip install -r requirements.txt`

## Run tests

* `./test_tokens.sh`
* `./transfer-pull.sh`
* `./transfer-pull.sh`
